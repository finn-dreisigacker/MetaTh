7.1 Data Import

The base analysis uses the pre-computed Hedges’ g file for Th2 at r = 0.5 (effect_size/Th2/pre_post/Th2_pre_post_r_050.csv). Sensitivity analyses can substitute the r = 0.3 / 0.4 / 0.6 / 0.7 files in the same folder. The pipeline column names are mapped to the analysis conventions (VE_g, Sex, Age, BMI) by load_es().

Code 
fullData <- load_es("book/data/effect_size/Th2/pre_post/Th2_pre_post_r_050.csv")

7.2 Primary Meta-Analysis

metagen() fits a random-effects model on the pre-computed effect sizes (g, SE_g) using the Hedges–Olkin (HE) τ² estimator and the Knapp–Hartung small-sample correction (hakn = TRUE). Kostrzewa-Nowak et al. (2019) is removed before pooling and the model re-fitted; metafor::rma() cross-validates.

  • Full Model {meta}
  • Full Model {metafor}
  • Adapted Model {meta}
Code 
m3 <- meta::metagen(TE = g, seTE = SE_g, data = fullData, studlab = paste(Study),
                    random = TRUE, method.tau = "HE", fixed = FALSE,
                    hakn = TRUE, prediction = TRUE, sm = "SMD")

7.3 Outlier Detection

Leave-one-out standardised residuals are computed via metaoutliers(), which is passed the variance (m3$seTE^2). Studies with |z| > 3 are flagged as outliers; |z| > 1.96 warrants inspection.

Code 
outlier <- metaoutliers(y = m3$TE, s2 = m3$seTE^2, model = "RE")
stdres <- outlier$std.res
Code 
fullData |> select(Short_reference, g) |> mutate(std_res = outlier$std.res) |> nice_table()
Code 
plot_stdres(stdres)
Caution

One outcome has standardized residuals around -2 (Kostrzewa-Nowak et al. (2019)).

7.4 Funnel Plot & Influential Cases

The contour-enhanced funnel plot shows study estimates against their standard errors; shaded regions mark significance zones (α = 0.10, 0.05, 0.01). The dfbetas diagnostic from metafor::rma.mv() identifies influential cases — distinct from statistical outliers and not necessarily excluded.

Code 
funnel_ce(m3, ref = 0.3214, xlim = c(-1, 2), ylim = c(0.7, 0))
Code 
overall_metafor <- rma.mv(g, VE_g, tdist = TRUE, data = fullData)
df_betas <- dfbetas(overall_metafor)
fullData |> select(Short_reference, g) |> mutate(dfbetas = df_betas) |> nice_table()
Tip

No influential case.

7.6 Asymmetry Test

Egger’s regression test (method = "linreg") evaluates funnel asymmetry as a proxy for small-study/publication effects (one-tailed, p < .10). Where asymmetry is suggested, a trim-and-fill analysis estimates potentially missing studies.

Code 
metabias(m3, method = "linreg")
Linear regression test of funnel plot asymmetry

Test result: t = 1.17, df = 15, p-value = 0.2595
Bias estimate: 2.2167 (SE = 1.8916)

Details:
- multiplicative residual heterogeneity variance (tau^2 = 3.2814)
- predictor: standard error
- weight:    inverse variance
- reference: Egger et al. (1997), BMJ
Note

No asymmetry -> no trim-and-fill

## Data Import

::: infobox
The base analysis uses the pre-computed Hedges' *g* file for Th2 at *r* = 0.5 (`effect_size/Th2/pre_post/Th2_pre_post_r_050.csv`). Sensitivity analyses can substitute the *r* = 0.3 / 0.4 / 0.6 / 0.7 files in the same folder. The pipeline column names are mapped to the analysis conventions (`VE_g`, `Sex`, `Age`, `BMI`) by `load_es()`.
:::

```{r th2-data}
fullData <- load_es("book/data/effect_size/Th2/pre_post/Th2_pre_post_r_050.csv")
```

## Primary Meta-Analysis

::: infobox
`metagen()` fits a random-effects model on the pre-computed effect sizes (*g*, *SE_g*) using the **Hedges–Olkin (HE)** τ² estimator and the **Knapp–Hartung** small-sample correction (`hakn = TRUE`). **Kostrzewa-Nowak et al. (2019)** is removed before pooling and the model re-fitted; `metafor::rma()` cross-validates.
:::

::: panel-tabset
### Full Model {meta}
```{r th2-main_meta_1}
m3 <- meta::metagen(TE = g, seTE = SE_g, data = fullData, studlab = paste(Study),
                    random = TRUE, method.tau = "HE", fixed = FALSE,
                    hakn = TRUE, prediction = TRUE, sm = "SMD")
print(m3, overall = TRUE)
```

### Full Model {metafor}
```{r th2-main-meta_2}
metafor::rma(yi = g, sei = SE_g, data = fullData, slab = paste(Study),
             method = "HE", test = "knha")
```

### Adapted Model {meta}
```{r th2-main-meta_3}
fullData <- drop_kostrzewa19(fullData)

                    random = TRUE, method.tau = "HE", hakn = TRUE,
                    prediction = TRUE, sm = "SMD", fixed = FALSE)
print(m3, overall = TRUE)
```

## Outlier Detection

::: infobox
Leave-one-out standardised residuals are computed via `metaoutliers()`, which is passed the **variance** (`m3$seTE^2`). Studies with \|*z*\| \> 3 are flagged as outliers; \|*z*\| \> 1.96 warrants inspection.
:::

```{r th2-outliers}
outlier <- metaoutliers(y = m3$TE, s2 = m3$seTE^2, model = "RE")
stdres <- outlier$std.res
```

```{r}
#| label: tbl-th2-imm-resid
#| tbl-cap: "Standardised residuals"
fullData |> select(Short_reference, g) |> mutate(std_res = outlier$std.res) |> nice_table()
```

```{r th2-outliers-plot}
plot_stdres(stdres)
```

::: {.callout-caution}
One outcome has standardized residuals around -2 (Kostrzewa-Nowak et al. (2019)).
:::

## Funnel Plot & Influential Cases

::: infobox
The contour-enhanced funnel plot shows study estimates against their standard errors; shaded regions mark significance zones (α = 0.10, 0.05, 0.01). The dfbetas diagnostic from `metafor::rma.mv()` identifies **influential cases** — distinct from statistical outliers and not necessarily excluded.
:::

```{r th2-funnel}
#| fig-width: 10
#| fig-height: 8
#| out-width: 100%
funnel_ce(m3, ref = 0.3214, xlim = c(-1, 2), ylim = c(0.7, 0))
```

```{r}
#| label: tbl-th2-imm-dfbetas
#| tbl-cap: "DFBETAS influence diagnostics"
overall_metafor <- rma.mv(g, VE_g, tdist = TRUE, data = fullData)
df_betas <- dfbetas(overall_metafor)
fullData |> select(Short_reference, g) |> mutate(dfbetas = df_betas) |> nice_table()
```

No influential case.

## Asymmetry Test

::: infobox
Egger's regression test (`method = "linreg"`) evaluates funnel asymmetry as a proxy for small-study/publication effects (one-tailed, *p* \< .10). Where asymmetry is suggested, a trim-and-fill analysis estimates potentially missing studies.
:::

```{r th2-egger}
metabias(m3, method = "linreg")
```

::: {.callout-note}
No asymmetry -> no trim-and-fill
:::
