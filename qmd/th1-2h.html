5.1 Data Import

The ~2 h window (effect_size/Th1/pre_post+2h/Th1_pre_post+2h_r_050.csv) contains few arms and no cytokine/BMI coverage, so it is reported for the main analysis, outlier check, funnel, forest, and asymmetry only — no moderator modelling is possible. No study is excluded a priori in this window.

Code 
fullData <- load_es("book/data/effect_size/Th1/pre_post+2h/Th1_pre_post+2h_r_050.csv")

## Data Import

::: infobox
The ~2 h window (`effect_size/Th1/pre_post+2h/Th1_pre_post+2h_r_050.csv`) contains few arms and no cytokine/BMI coverage, so it is reported for the main analysis, outlier check, funnel, forest, and asymmetry only — no moderator modelling is possible. No study is excluded a priori in this window.
:::

```{r th1-2h-data}
fullData <- load_es("book/data/effect_size/Th1/pre_post+2h/Th1_pre_post+2h_r_050.csv")
```

## Primary Meta-Analysis

::: panel-tabset

m1 <- meta::metagen(TE = g, seTE = SE_g, data = fullData, studlab = paste(Study),

print(m1, overall = TRUE)

## Outlier Detection

```{r th1-2h-outliers}
outlier <- metaoutliers(y = m1$TE, s2 = m1$seTE^2, model = "RE")
stdres <- outlier$std.res
```

```{r}
#| label: tbl-th1-2h-resid
#| tbl-cap: "Standardised residuals"
fullData |> select(Short_reference, g) |> mutate(std_res = outlier$std.res) |> nice_table()
```

```{r th1-2h-outliers-plot}
plot_stdres(stdres)
```


::: {.callout-note}
No outcome has standardized residuals above 3 or below -3.
:::


## Funnel Plot & Influential Cases

```{r th1-2h-funnel}
#| fig-width: 10
#| fig-height: 8
#| out-width: 100%
funnel_ce(m1, ref = 0.99, xlim = c(-1.5, 4), ylim = c(0.7, 0))
```

```{r}
#| label: tbl-th1-2h-dfbetas
#| tbl-cap: "DFBETAS influence diagnostics"
overall_metafor <- rma.mv(g, VE_g, tdist = TRUE, data = fullData)
df_betas <- dfbetas(overall_metafor)
fullData |> select(Short_reference, g) |> mutate(dfbetas = df_betas) |> nice_table()
```

::: notebox
One value below −1 and one above 1 were identified as influential cases. Neither is an outlier (Kostrzewa-Nowak & Nowak (2020a)a and Tsubakihara et al. (2012)); all outcomes are retained.
:::
