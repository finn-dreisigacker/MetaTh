4.1 Data Import

The base analysis uses the pre-computed Hedges’ g file for Th1 at r = 0.5 (effect_size/Th1/pre_post/Th1_pre_post_r_050.csv). Sensitivity analyses can substitute the r = 0.3 / 0.4 / 0.6 / 0.7 files in the same folder. The pipeline column names are mapped to the analysis conventions (VE_g, Sex, Age, BMI) by load_es().

Code 
fullData <- load_es("book/data/effect_size/Th1/pre_post/Th1_pre_post_r_050.csv")

4.2 Primary Meta-Analysis

metagen() fits a random-effects model on the pre-computed effect sizes (g, SE_g) using the Hedges–Olkin (HE) τ² estimator and the Knapp–Hartung small-sample correction (hakn = TRUE). After inspecting the full set, Kostrzewa-Nowak et al. (2019) was detected as very different across meta-analysis and while not influential on the overall effect, its values were unrealistic and brought substantial misleading variance.

  • Full Modell {meta}
  • Adapted Model {meta}
  • Adapted Model {metafor}

4.3 Outlier Detection

Leave-one-out standardised residuals are computed via metaoutliers(), which is passed the variance (m1$seTE^2). Studies with |z| > 3 are flagged as outliers; |z| > 1.96 warrants inspection.

Code 
outlier <- metaoutliers(y = m1$TE, s2 = m1$seTE^2, model = "RE")
stdres <- outlier$std.res
Code 
fullData |> select(Short_reference, g) |> mutate(std_res = outlier$std.res) |> nice_table()
Code 
plot_stdres(stdres)
Important

No outcome has standardized residuals above 3 or below -3.

4.4 Funnel Plot & Influential Cases

The contour-enhanced funnel plot shows study estimates against their standard errors; shaded regions mark significance zones (α = 0.10, 0.05, 0.01). The dfbetas diagnostic from metafor::rma.mv() identifies influential cases — distinct from statistical outliers and not necessarily excluded.

Code 
funnel_ce(m1, ref = 0.99, xlim = c(-1.5, 4), ylim = c(0.7, 0))
Code 
overall_metafor <- rma.mv(g, VE_g, tdist = TRUE, data = fullData)
df_betas <- dfbetas(overall_metafor)
fullData |> select(Short_reference, g) |> mutate(dfbetas = df_betas) |> nice_table()

One value was below −1 and one above 1, so both were identified as influential cases. However, neither is an outlier (Kostrzewa-Nowak & Nowak (2020a)a and Tsubakihara et al. (2012)). Accordingly, all outcomes are retained, and the sources of heterogeneity are examined in the moderator analysis.

4.6 Asymmetry Test

Egger’s regression test (method = "linreg") evaluates funnel asymmetry as a proxy for small-study/publication effects (one-tailed, p < .10). Where asymmetry is suggested, a trim-and-fill analysis estimates potentially missing studies.

  • Egger´s regression test
  • Trim and Fill
Code 
metabias(m1, method = "linreg")

- multiplicative residual heterogeneity variance (tau^2 = 6.0667)

## Primary Meta-Analysis

::: infobox
`metagen()` fits a random-effects model on the pre-computed effect sizes (*g*, *SE_g*) using the **Hedges–Olkin (HE)** τ² estimator and the **Knapp–Hartung** small-sample correction (`hakn = TRUE`). After inspecting the full set,
**Kostrzewa-Nowak et al. (2019)** was detected as very different across meta-analysis and while not influential on the
overall effect, its values were unrealistic and brought substantial misleading variance.
:::

::: panel-tabset

print(m1, overall = TRUE)

## Funnel Plot & Influential Cases

::: infobox
The contour-enhanced funnel plot shows study estimates against their standard errors; shaded regions mark significance zones (α = 0.10, 0.05, 0.01). The dfbetas diagnostic from `metafor::rma.mv()` identifies **influential cases** — distinct from statistical outliers and not necessarily excluded.
:::

```{r th1-funnel}
#| fig-width: 10
#| fig-height: 8
#| out-width: 100%
funnel_ce(m1, ref = 0.99, xlim = c(-1.5, 4), ylim = c(0.7, 0))
```

```{r}
#| label: tbl-th1-imm-dfbetas
#| tbl-cap: "DFBETAS influence diagnostics"
overall_metafor <- rma.mv(g, VE_g, tdist = TRUE, data = fullData)
df_betas <- dfbetas(overall_metafor)
fullData |> select(Short_reference, g) |> mutate(dfbetas = df_betas) |> nice_table()
```

::: notebox
One value was below −1 and one above 1, so both were identified as influential cases. However, neither is an outlier (Kostrzewa-Nowak & Nowak (2020a)a and Tsubakihara et al. (2012)). Accordingly, all outcomes are retained, and the sources of heterogeneity are examined in the moderator analysis.
:::
